# VitalChain

A Vite + React dashboard prototype for pharmaceutical cold chain monitoring.

## Available scripts

- `npm run dev` — start the development server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build locally

## Project structure

- `index.html` — app entry HTML
- `src/main.jsx` — React entry point
- `src/App.jsx` — dashboard UI
- `src/index.css` — global styles
- `src/App.css` — dashboard component styles
- `vite.config.js` — Vite configuration

## Notes

- This workspace uses Vite with React 18.
- Run `npm install` first if dependencies are missing.
